How long until first paying client?

Realistic numbers based on what other operators achieve with the same stack:

  • • Day 1–2: buy domain, set up mailbox, enter SMTP keys.
  • • Day 3–14: warmup. Send 2–5 emails/day, no client work yet.
  • • Day 15: add first paying client.
  • • Day 15–28: nightly hunt + morning brief + manual outreach. Expect ~3–8% reply rate at this stage.
  • • Day 28: first HOT reply → forward to client → first invoice.
